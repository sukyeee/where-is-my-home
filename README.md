<h1 align="center">Where Is My House</h1>

## Features

- Vue / Spring Project
  - 로그인, 회원가입, 마이페이지
  - 주소검색
  - 북마크
  - 공지사항
  - 이벤트
  - 질문게시판

## API

- 카카오지도: https://apis.map.kakao.com/

## Tech

- ES6
- Vue
- SCSS
- axios
- Vue-Router
- Vuex
- mySql
